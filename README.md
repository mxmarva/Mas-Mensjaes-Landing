# Mas Mensajes — Landing Page

Sitio estático para [Mas Mensajes](https://www.mas-mensajes.com), optimizado para **GitHub Pages**.

## Estructura

```
├── index.html
├── aviso-de-privacidad.html
├── terminos-y-condiciones.html
├── css/styles.css
├── js/main.js
└── assets/images/
```


## Enlaces de la aplicación

- Login: https://app.mas-mensajes.com/login
- Registro: https://app.mas-mensajes.com/register

## Desarrollo local

Abre `index.html` en el navegador o usa un servidor estático:

```bash
npx serve .
```
